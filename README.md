# Chop.
Chop is a frontend portfolio project  a dark, cinematic Nigerian food and recipe website built with HTML and CSS 
Chop. 
Nigerian food, beautifully served.
A clean, dark, timeless Nigerian food website where people can discover dishes and learn how to prepare them. Built because every Nigerian food site I found was either cluttered, poorly designed, or didn't represent our cuisine the way it deserves.
